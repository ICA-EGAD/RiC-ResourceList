Parser for diagrams created using the RiC draw.io shape library

  • Richard Williamson
  • 2024-04

  • A Python script which takes the underlying XML of a draw.io diagram in which the RiC shape library has been used, and outputs OWL individuals whose types are classes in RiC-O, and which are related by RiC-O properties. In other words, it creates a formal RiC-O graph from the informal one in draw.io.

  • GitHub repository
  • Available in: English
  • RiC-O 1.0
  • It is planned to add support for RiC-CM, and also to include Relation classes in RiC-O, which are currently omitted. Output formats other than OWL are also planned.
