Waiting for RiC

  • Richard Dancy
  • 2024-11-01

  • RiC’s authors foresee a gradual transition period, in which the existing standards continue to be used as both archivists and software developers find their way with the new standard. The purpose of this communication note is to help clarify some of the issues likely to be encountered along that way: What is RiC, how does it differ from previous standards, and what does it all mean for Canadian archivists still lumbering along with the Rules for Archival Description (RAD)? This note provides some background, exposition, and commentary, and it suggests some practical ways by which archivists can starting using RiC without adopting it (in the absence of RiC software).
